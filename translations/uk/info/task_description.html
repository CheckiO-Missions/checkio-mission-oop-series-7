7.1. Додай атрибут fuel_used у метод __init__ класу Car, не змінюючи його аргументів, і признач йому значення 0 літрів.

7.2. Додай атрибут fuel_consumption у метод __init__ класу Car, передаючи його як аргумент методу зі значенням за замовчуванням 7 (літрів/100 км).

7.3. Додай метод drive до класу Car, який приймає аргумент distance типу ціле число (int) (км). Якщо екземпляр класу було запущено, збільш fuel_used із відповідним значенням і відобрази повідомлення "Currently driven {distance} km, total fuel used - {fuel_used} l". Якщо екземпляр не було запущено, відобрази "Start the car before driving!".

7.4. Перепиши метод для класу ElectricCar, не збільшуй fuel_used (оскільки електромобіль не використовує паливо), зміни успішне повідомлення на "Currently driven {distance} km on electric motor", залиш повідомлення про невдачу без змін.
