7.1. Proszę dodać atrybut fuel_used wewnątrz metody __init__ klasy Car bez zmiany jej argumentów i przypisać mu wartość 0 litrów.

7.2. Proszę dodać atrybut fuel_consumption wewnątrz metody __init__ klasy Car, przekazując go jako argument metody z domyślną wartością 7 (litry/100 km).

7.3. Proszę dodać metodę drive do klasy Car, która przyjmuje argument distance typu int (km). Jeśli instancja klasy została uruchomiona - proszę zwiększyć fuel_used o odpowiednią wartość i wyświetlić komunikat "Currently driven {distance} km, total fuel used - {fuel_used} l". Jeśli instancja nie została uruchomiona, proszę wyświetlić "Start the car before driving!".

7.4. Proszę przepisać metodę dla klasy ElectricCar, nie inkrementować fuel_used (ponieważ samochód elektryczny nie zużywa paliwa), zmienić komunikat o powodzeniu na "Currently driven {distance} km on electric motor", komunikat o niepowodzeniu pozostawić bez zmian.
